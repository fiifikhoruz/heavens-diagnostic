Technical Architecture

Next.js 16 + Supabase + TypeScript, 44 source files, 11 migrations

Frontend (Next.js)

  • App Router with TypeScript
  • Tailwind CSS styling
  • Role-based navigation
  • Auto-save drafts (IndexedDB)
  • Retry queue (localStorage)
  • 20-minute session timeout

Backend (Supabase)

  • PostgreSQL with RLS
  • Auth with session tokens
  • 11 SQL migrations
  • DB triggers for workflow
  • Views for analytics
  • RPC functions

Security

  • Rate limiting (10 req/15min)
  • Account lock (5 fails = 30min)
  • Sensitive data masking (HIV etc)
  • Result revision audit trail
  • Admin activity logging
  • No-delete policy on results
